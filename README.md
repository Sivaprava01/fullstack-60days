# Fullstack 60 Days Build Challenge

This repository contains my 60-day fullstack development journey.

## Description
This is my repo containing 60 projects
first 30 days with mostly frontend and the next 30 days with backend 
this is just to brush up my frontend skills and learn backend on the go.

## Tech Stack
- Frontend: HTML, CSS, JavaScript, React
- Backend: Java, Spring Boot, REST APIs
- Database: MongoDB
- Version Control: Git & GitHub

## Structure
- Each week has its own folder
- Each day contains one real-world feature

Goal: Build consistently and understand deeply.
